Today Tab (Daily Planner)

Today View
• Date header + progress ring
• Vertical timeline (scrollable)
• Task cards (icon · title · time)
• One-tap complete
• "No tasks" friendly empty state
• FAB → add task sheet
Add / Edit Task
• Title input
• Icon picker (300 icons)
• Color chip row
• Start time + duration
• Reminder toggle
• Repeat rule picker
• Notes field
• Save / Delete
Week View
• 7-column header strip
• Per-day completion dot
• Tap day → Today View for that day
• Swipe left/right to navigate weeks

Habits Tab

Habit List
• Today's habits with progress
• Streak badge per habit
• Quick-complete tap
• Up to 5 habits (MVP)
• FAB → add habit
Add / Edit Habit
• Title + icon + color
• Type: Boolean / Count / Duration
• Frequency: daily / specific days / N×week
• Optional reminder time
• Save / Archive
Habit Detail
• Current streak + longest streak
• 30-day heatmap
• This week % completion
• Log history list
• Edit / Archive buttons

Me Tab (Profile)

Profile / Settings
• Avatar + display name
• Account status (local / signed in)
• Sign in / Sign out
• Notification settings
• Theme toggle (Light / Dark)
• Export data (CSV)
• Delete account
• App version + support link
