Sync strategy (MVP): Online-first. Flutter writes to Drift cache immediately for instant UI feedback, then fires an API call. On app start / pull-to-refresh, latest server state overwrites local cache. Conflict resolution: server wins.
Auth: Email/password + Google OAuth via Spring Boot. Server issues short-lived JWT (15 min) + refresh token (30 days) stored in Android Keystore. Anonymous local mode supported — account creation is optional.
